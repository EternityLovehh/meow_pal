# meow_pal — 桌面元气猫设计文档

**日期:** 2026-07-17
**类型:** 全新独立项目(与 decard-android 无关)
**技术栈:** Flutter Desktop(跨平台 macOS + Windows)

## 1. 定位

一只用代码"漂"在桌面上、元气满满的猫,陪你工作,到点用可爱的方式提醒你喝水和起身。核心是**情绪价值**:陪伴 + 恰到好处的关心,不打扰。

## 2. 第一版范围(MVP)

**做(In):**
- 🐱 一只 Rive 猫,常驻桌面、永远置顶、背景透明、可拖动
- 👆 基础互动:点它(开心)、拖它(移动)、久置不理(自娱)
- ⏰ 两个提醒:喝水、站立(纯桌面表现,不发系统通知、不出声)
- 🌙 时间感知:深夜仍开着 → 冒气泡关心一句
- ⚙️ 设置:各提醒开关、间隔调节、深夜关心开关
- 🖥️ 跨平台:macOS + Windows
- 🔻 系统托盘:右键退出 / 打开设置

**不做(Out):**
- 忙碌/前台应用感知(只到"时间 + 闲置")
- 投喂/养成/成长、多宠物、皮肤商店
- 云同步、账号
- AI 对话(只说预设短句)
- 开机自启

## 3. 形象与美术

- **动物:** 猫(元气人格主选);水豚为备选(若 Rive 素材更佳)
- **美术来源:** Rive 社区现成角色(rive.app),挑选 + 在 Rive 编辑器可视化调整配色/细节,导出 `.riv` 接入 Flutter。**不手绘、不 AI 出图。**
- **要求:** 自带 idle / 眨眼 / 点击反应等状态机;好看、中性(非少女向)、授权允许使用
- **理由:** 唯一能同时满足「好看 + 会动 + 不会画 + 跨平台」的路径

## 4. 人格

**元气陪伴:** 活泼、正向、俏皮,会给小鼓励。台词为短句,第一版中文写死在台词库,每类事件从 3-5 句中随机挑一句避免重复。

## 5. 技术架构

```
meow_pal/
├─ 窗口层    透明/无边框/置顶窗口 + 拖动 + 点击穿透  (window_manager)
├─ 形象层    Rive 猫 + 状态机(idle/开心/提醒/打盹/关心)
├─ 行为层    ├ 提醒调度器(喝水/站立定时器)
│            ├ 感知器(闲置时长 + 当前时间)
│            └ 人格台词库(按事件挑一句)
├─ 设置层    设置窗口 + 本地持久化
└─ 系统集成  系统托盘(退出/打开设置)
```

**模块边界:** 各层职责单一、经明确事件通信、可独立测试。
- 提醒调度器:只负责"到点发出 `WaterReminder` / `StandReminder` 事件",不关心表现
- 形象层:只负责"收到事件 → 播对应 Rive 动画",不关心事件来源
- 台词库:输入事件类型 → 输出一句台词
- 行为逻辑(调度、感知、选词)可脱离 UI 单元测试

## 6. 关键技术风险:点击穿透(click-through)

猫是不规则形状、窗口是方形,透明区域必须让鼠标穿透到底层。

- **方案:** `window_manager` 的 `setIgnoreMouseEvents(true/false)`。默认整窗穿透;鼠标 hover 到"猫身上"时临时关闭穿透(可点/可拖),移开再开启。桌宠通用做法。
- **风险前置:** 开工第一步先写最小 demo(透明置顶窗 + 一个圆),验证"透明区穿透、圆上可点可拖"在 macOS + Windows 均成立,再往下堆功能。

## 7. 行为规格

| 触发 | 动作(Rive 状态) | 气泡台词(占位,3-5 句轮换) |
|---|---|---|
| 常态待机 | idle:呼吸/偶尔甩尾/眨眼 | (通常不说话) |
| 点它一下 | 开心:蹦/扭 | "嗯哼~""在的在的!""摸鱼被我抓到啦😼" |
| 拖动它 | 被拎起:拉伸/腿晃 | "喵呜——!""放我下来啦~" |
| 久坐不理(闲置 > 阈值) | 打盹/东张西望/舔爪 | (安静,偶尔)"…zzz""还在吗?" |
| 喝水提醒 | 蹦跳 + 水杯气泡 | "该喝水啦!""补个水,冲鸭💧" |
| 站立提醒 | 伸懒腰/原地跳 | "站起来动动~""久坐会变石雕哦!" |
| 深夜仍开着 | 关心:歪头/揉眼 | "这么晚啦,早点睡呀~🌙" |
| 提醒后你点它回应 | 更开心:转圈 | "耶!你最棒!""就知道你可以😽" |

## 8. 默认参数(均可在设置中修改)

| 参数 | 默认值 |
|---|---|
| 喝水间隔 | 45 min |
| 站立间隔 | 30 min |
| 闲置判定 | 5 min |
| 深夜阈值 | 23:30 |
| 台词语言 | 中文(写死,后续再抽多语言) |

**设置窗口内容:** 各提醒开关 + 三个间隔滑块 + 深夜关心开关。
